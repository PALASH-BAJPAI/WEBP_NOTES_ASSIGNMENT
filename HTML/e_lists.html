Unordered Lists

The unordered list starts with the <ul> tag. Each list item is listed between the starting and closing tag of <li> . This list uses bullets rather than a conventional numbering system. The style of bullets can be given through the type attribute. Let's look at an example of the same.

  • Apple
  • Banana
  • Orange
  • Mango

Ordered Lists

The ordered list starts with the <ol> tag. Each item is enclosed is <li> tag. This type of list has a systematic numbering system like numeric, roman, alphabetic, etc. The type of the same can be changed by the type attribute. Let's look at the same example given above as an ordered list.

  1. Apple
  2. Banana
  3. Orange
  4. Mango
