Forms

Forms are used to take input from the user. The input can be of various types like text, password, radio button, checkbox, etc. The <form> tag is used to create a form.
The <input> tag is used to create the input fields. The type attribute is used to specify the type of input field. The value attribute is used to specify the default value of the input field. Notes:

  • Notice that each input field must have a name attribute to be submitted. If the name attribute is omitted, the value of the input field will not be sent at all.
  • The action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.
  • The method attribute specifies the HTTP method to be used when submitting the form data. The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post"). The default HTTP method when submitting form data is GET.
