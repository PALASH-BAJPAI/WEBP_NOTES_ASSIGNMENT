Selectors

  1. Element selector

    2. Selects based on element name.

  2. Class selector

    3. Selects based on class name, referenced as .class. Classes of certain elements can be referenced by element.class.

  3. ID selector

    4. The id of an element is unique within a page, so the id selector is used to select one unique element and is referenced as a #id.

  4. Universal selector

    5. Selects all elements

  5. Grouping selector

    6. Selects all the elements with the same style definitions, separating each selector by a comma.
