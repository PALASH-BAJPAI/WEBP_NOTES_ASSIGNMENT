Flex Container

The flex container becomes flexible by setting the display property to flex. The flex container is considered to be the parent element of all flex elements inside.

The flex container properties are:

  • flex-direction

    • Defines in which direction the container wants to stack the flex items. The values are:

    • row

      • left-to-right arrangement

    • row-reverse

      • right-to-left arrangement

    • column

      • top-to-bottom arrangement

    • column-reverse

      • bottom-to-top arrangement

  • flex-wrap

    • Specifies whether the flex items should wrap or not. The values are:

    • nowrap

      • all flex items will be on one line. This is the default

    • wrap

      • flex items will wrap onto multiple lines, from top to bottom

    • wrap-reverse

      • flex items will wrap onto multiple lines, but in reverse order

  • flex-flow

    • A shorthand property for the flex-direction and the flex-wrap properties. The default value is row nowrap.

  • justify-content

    • Used to align the flex items. The values are:

    • flex-start

      • items are positioned at the beginning of the container. This is the default

    • flex-end

      • items are positioned at the end of the container

    • center

      • items are positioned at the center of the container

    • space-between

      • items are positioned with space between the lines

    • space-around

      • items are positioned with space before, between, and after the lines

  • align-items

    • used to align the flex items.

    • stretch

      • stretch to fill the container (this is default)

    • flex-start

      • items are positioned at the beginning of the container

    • flex-end

      • items are positioned at the end of the container

    • center

      • items are positioned at the center of the container

    • baseline

      • items are positioned at the baseline of the container

  • align-content

    • Used to align the flex lines.

    • stretch

      • lines stretch to take up the remaining space (this is default)

    • flex-start

      • lines are packed at the top of the container

    • flex-end

      • lines are packed at the bottom of the container

    • center

      • lines are packed at the center of the container

    • space-between

      • lines display with equal space between them

    • space-around

      • lines display with equal space around them

Perfect Centering

Set both the justify-content and align-items properties to center, and the flex item will be perfectly centered.

Flex Items

Flex items are the children of a flex container. Flex items can be laid out in any direction and in any order.
Flex items can also "flex" their sizes, either growing to fill unused space or shrinking to avoid overflowing the parent.

The flex item properties are:

  • order

    • Specifies the order of the flex items. The default value is 0, so the first item will be first in the display.

  • flex-grow

    • Specifies how much the item will grow relative to the rest of the flexible items inside the same container. The default value is 0, so the item will not grow at all.

  • flex-shrink

    • Specifies how the item will shrink relative to the rest of the flexible items inside the same container. The default value is 1, so the item will shrink equally to its peers.

  • flex-basis

    • Specifies the initial length of a flexible item. The default value is auto, which sizes the item based on its content.

  • flex

    • A shorthand property for the flex-grow, flex-shrink, and the flex-basis properties.

  • align-self

    • Specifies the alignment for the selected item inside the flexible container. The default value is auto, which will use the default alignment set by the container. This property overrides the container's align-items property.
