Clear and Clearfix

The clear property specifies what should happen with the element that is next to a floating element. The clear property can have one of the following values:

  • none
  • left
  • right
  • both
  • inherit

When clearing floats, the value of the clear to the float.

The clearfix property solves the overflow of a taller element in the smaller container element.
The overflow: auto clearfix works well as long as you are able to keep control of your margins and padding (else you might see scrollbars)
