Display

Default for div is block
Span: For span default display is inline Span: We can not change width and height of span element Span: If we want to set height and width of span element, make it inline-block


Flex

The flex layout allows responsive elements within a container to be automatically arranged depending on viewport size. Display:flex
Default flex-direction is row, but can change it to column justify-content: align in main axis(default horizontal), align-item: align in cross-axis The "flex-wrap: wrap;" specifies that the flex items will wrap if necessary
