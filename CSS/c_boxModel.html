A mixed border. The border conditions are defined in the order of top, right, bottom, left.

Background

  • By default, the image is repeated so it covers the entire element.
  • If the image is to be repeated only horizontally, the background-repeat can be set to repeat-x. Similarly repeat-y for vertical repetition.
  • The background-position property is used to specify the position of the background image.
  • The background-attachment property specifies whether the background image should scroll or be fixed
  • In short, background can be used with the order of color, image, repeat, attachment and position instead of all the above codes.
